기초가 탄탄한, 창의적인, 항상 발전하는 신입 웹퍼블리셔 오주영입니다.

저는 사용자 입장에서 코딩을 합니다.
웹 표준을 준수한 크로스 브라우징을 바탕으로
사용자가 더 편리하게 웹페이지를 이용할지 항상 고민합니다.

또한 홈페이지를 소유한 관리자의 입장에서 코딩을 합니다.
검색엔진 최적화 작업을 하여 더 많은 사용자가 웹사이트에 접속하도록 유도하고 유지 보수를 쉽게 할 수 있도록 작업합니다.

마지막으로 창의적인 코딩을 만들려고 노력합니다.
CSS와 JAVASCRIPT를 사용해 다른 웹사이트와 차별점을 만들려 노력합니다.

저는 코딩을 좋아합니다. 즐겁게 일을 하다 보니 더 잘하고 싶어집니다
그러다 보니 어느새 목표를 하나씩 달성하고 있는 저를 발견했습니다.

계속 발전하고 고민하면서
이 분야에서 최고가 되고 싶습니다.

기본기가 탄탄한
신입 웹 퍼블리셔

  • UI Development

    웹 퍼블리싱 HTML, CSS, JS을 빠르게 이해하고 해석하며 잘 다룹니다.
    유지 보수를 고려한 코드 작성, 협업을 위한 가이드 제작 및 자바스크립트 커스터마이징이 가능합니다.

  • Markup

    웹 표준을 바탕으로 꼼꼼한 시멘틱 마크업을 작성합니다.
    크로스 브라우징과 검색엔진 최적화(SEO)를 통해 더 많은 사용자들에게 접근할 수 있도록 합니다.

  • Communication

    디자이너, 개발자와 항상 소통하며 공유합니다.
    의도를 이해하고 존중하며 '함께' 좋은 결과물을 만듭니다.

  • Improve

    코드를 만들기 위해 계속 공부합니다.
    Sass, Vue, React 등 빠르게 변하는 웹 트렌드를 따라 공부하며 발전하려고 노력합니다.

  • For Users

    사용자 입장에서 생각하고 고민하며 작업합니다.
    혼자만의 생각이 아닌 다양한 시각을 얻기 위해 팀원, 웹 퍼블리셔들과 경험을 공유하고 배움의 자세로 경청합니다.
